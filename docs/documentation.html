‚ùì Frequently Asked Questions

What is HashTrace?

HashTrace is an enterprise-grade audit trail platform that provides cryptographically verified event tracking with complete compliance auditing. It ensures tamper-proof audit trails using SHA-256 hash chains and Ed25519 signatures.

How fast is HashTrace?

HashTrace can process events in less than 5 milliseconds with a throughput of 10,000+ events per second. This makes it suitable for high-volume audit trail requirements.

What compliance standards does HashTrace support?

HashTrace supports SOC 2, HIPAA, PCI-DSS, and GDPR compliance standards. It includes built-in compliance auditing and can be configured with custom compliance rules.

Can I deploy HashTrace on-premises?

Yes, HashTrace can be deployed on-premises using Docker or Kubernetes. See the Deployment Guide for detailed instructions.

What SDKs are available?

HashTrace provides official SDKs for Node.js and Python, along with a REST API and CLI tool. Additional SDKs can be developed using the REST API.

How do I integrate HashTrace with my application?

You can integrate HashTrace using the official SDKs, REST API, or webhooks. See the API Reference and SDK documentation for detailed integration examples.

Is HashTrace open source?

Yes, HashTrace is open source under the Apache 2.0 license. You can find the source code on GitHub at https://github.com/BMKME/hashtrace

How can I contribute to HashTrace?

We welcome contributions! Please see the Contributing Guide on GitHub for information about development setup, code standards, and the pull request process.
